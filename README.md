A modern e-commerce platform built with microservices architecture using .NET 9

## Services

- CRUD operations
- filtering
- PostgreSQL (JSON documents) with Carter
- .Net 9 - Hybrid caching(preview) (Redis + In-Memory)
- CQRS pattern with MediatR

## Tech Stack
- .NET Core
- Docker
- PostgreSQL
- Redis
- MediatR
- FluentValidation
- Mapster
- Carter
    
## Features

- Exception handling
- Request validation
- Health monitoring
- Logging
- Docker containerization
- CQRS pattern
- Distributed caching

## Getting Started
1. Run:
```bash
docker-compose up -d
```
